# Open Beta Notes

LLR を BRAT で試す人向けの短い案内。
「完成品を評価する」より、「日々の運用でどこが噛み合うか」を見るためのメモです。

## いま試してほしいこと

- デイリーノート上で `Toggle Task` を使って開始と完了を記録する
- `Adjust Time (1m)` で軽い時刻補正が自然にできるか確認する
- `Insert Routine` で `routine/` から当日分を差し込む
- `Open Summary View` で、その日の流れが追いやすいかを見る
- モバイルでは短押し / 長押しが素直に使えるかを見る

短時間で回したい時は [Beta Checklist](beta-checklist.md) を使う。

## いま期待してよい範囲

- Markdown の見た目を崩さずに、開始時刻と完了時刻を残せる
- 実行中タスクと残見積りを Summary View で把握できる
- ルーチンを毎日の作業面に差し込める

## まだ粗さが残っていてよい範囲

- すべての入力ゆらぎをきれいに吸収すること
- Sidebar 上での直接編集
- 日次アーカイブやロールオーバーの自動化
- Community Plugins 公開前提の最終ポリッシュ

## フィードバックで助かること

- どの操作で止まったか
- どの Markdown 行で意図どおりに変換されなかったか
- モバイルで押しづらい、分かりづらい挙動
- Summary View で「いま何を見たいか」と実際に見えるもののズレ
