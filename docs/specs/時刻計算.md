# TimeCalculator Logic Specification

## 1. Overview
時刻計算、所要時間算出、および日付境界の判定を行う純粋関数群。

## 2. Interface

```typescript
interface TimeInput {
  now: Date;
  startTime?: string; // "09:00"
  estimateMin?: number;
}

interface TimeOutput {
  endTime: string;    // "09:45"
  durationMin: number;
}
```

## 3. Logic Requirements

### A. Add Duration
- **Input**: `startTime: "09:00"`, `duration: 45`
- **Output**: `endTime: "09:45"`
- **Edge Case**:
    - `23:30` + `60m` -> `00:30` (翌日だが、表示は `00:30` または `24:30` ※設定依存、今回は24h表記でロールオーバー)

### B. Calculate Duration
- **Input**: `start: "09:00"`, `end: "09:45"`
- **Output**: `45` min
- **Edge Case**:
    - 日付またぎ: `start: "23:50"`, `end: "00:10"` -> `20` min (マイナスにならないこと)

### C. Estimate from Link
- **Input**: `[[Routine (45m)]]` (Obsidian Link)
- **Output**: `45`
- **Logic**:
    - リンクテキスト内に `(XXm)` があればそれを優先。
    - なければ YAML (Frontmatter) を参照（これは別モジュールだが、I/Fとして定義）。
