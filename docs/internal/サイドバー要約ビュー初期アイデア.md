# Sidebar Summary View (Idea)

サイドバーに「今日のタスク」と「合計見積り時間」のエッセンスだけを表示する、コンパクトなビューの提案。

## 1. コンセプト
現在開いているデイリーノート（`today.md` 等）から、重要なサマリー情報を抽出してサイドバーに常駐させる。これにより、メインエディタで長いタスクリストをスクロールしなくても、常に「ゴール」と「現在の位置」を把握できるようにする。

## 2. 表示項目
必要最小限の情報に絞る：

- **今日の残り見積り合計**: `today.md` に残っている未完了タスクの合計見積り時間。
- **今後の予定（直近5〜10件）**: 
    - まだ開始していないタスクの見出し。
    - **各タスクごとの見積り時間**。
- **現在のタスク（もしあれば）**: 実行中のタスクがある場合、それを最上部に強調表示。

## 3. UI イメージ
```
[ project-tc Summary ]
----------------------
Remaining: 3h 15m
Finish at: 18:45

> [RUNNING] ブログ下書き (30m)

- [ ] メール返信 (15m)
- [ ] MTG準備 (45m)
- [ ] 週次報告作成 (60m)
- [ ] 読書 (30m)
- [ ] 散歩 (20m)
----------------------
```

## 4. 特徴・メリット
- **気が散らない**: 余計なメモやリンクを省き、タスク名と時間だけに集中できる。
- **モバイル対応**: スクリーンが狭い環境でも、サイドバー（右パネル）でサマリーを確認できる。
- **リアルタイム更新**: エディタでタスクを開始・完了したり、見積りを書き換えたりすると、サイドバーの数値も同期して更新される。

## 5. 優先順位
- **優先度**: 低 (Phase 3 以降)
- **ステータス**: アイデア段階 / 計画中
